Las
  prácticas a realizar serán las siguientes:



Práctica 1. Prototipado rápido. Se
  aprenderá a preparar prototipos mediante tecnología
  FDM. 



Práctica 2. Simulación de un proceso por
  deformación plástica. Se realizará
  un modelo numérico para predecir fuerzas y deformaciones
  en una lámina de PVC sometida a un proceso de
  conformado.



Práctica 3. Simulación de un proceso de
  arranque de material. Se simulará
  numéricamente un proceso de mecanizado.



Práctica 4. Aplicación de técnicas
  de machine learning a procesos de
  fabricación. Se aplicarán varias
  técnicas de clusterización y optimización a
  datos reales de procesos de fabricación y se
  interpretarán los resultados.



Práctica 5. Medición con sistema
  electrónico. Se aprenderá a montar y
  calibrar un sistema de medida con una cadena
  electrónica
